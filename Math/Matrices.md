## Properties

A + B = B + A #associative-property
(A + B) + C = A + (B + C) #communitaive-property
$\alpha$(A + B) = $\alpha$A + $\alpha$B #distributive-property
$\alpha + \beta$(A + B) = $\alpha$A + $\beta$B #distributive-property 
($\alpha\beta$)(A) = $\alpha$($\beta$A)
$(A^T)^T=A$
